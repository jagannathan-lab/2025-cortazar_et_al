# 2025-cortazar_et_al
Code to analyze data and make figures in the paper titled "Genomic stop codon scanning reveals quantitative principles of nonsense-mediated mRNA decay" Preprint: https://www.biorxiv.org/content/10.64898/2025.12.20.695734v1

Code description:
- python scripts are used within .sh scripts
- SeqPrep.sh and Bowtie.sh to be run first
- Relative_mRNA_expression.sh and Calculate_NMD_efficiency.sh can be done next in any order
